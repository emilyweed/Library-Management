## Overview
The `index.php` file is the main entry point for "The Book Burrow." It displays recommended books and available media by fetching data from the Open Library API.

## Prerequisites
1. A local PHP server (e.g., XAMPP, WAMP, or built-in PHP server).
2. An internet connection to fetch data from the Open Library API.

## Running the File
1. Place the `1_code` folder in your server's root directory (e.g., `htdocs` for XAMPP).
2. Start your server.
3. Access the file by navigating to `http://localhost/1_code/index.php` in your browser.

## Features
- Recommended Books: Displays 10 books related to "Harry Potter."
- Available Media: Displays 10 books based on a predefined query ("available").
- Action Buttons: Allows users to "Borrow" or "Hold" books (currently for demonstration).
